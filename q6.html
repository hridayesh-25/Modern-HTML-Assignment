HTML VS HTML5

Feature HTML HTML5
Document structure Utilizes elements like head, body, title, etc.. Enhanced semantic structure with new elements like header, nav, section, footer, article, etc..
Media support Limited multimedia support. Requires plugins like flash for advanced media handling. Built-in support for audio and video elements without requiring third-party plugins.
Form Elements Basic form elements like input, textarea, select, etc. Introduces new input types like email, tel, date, range, etc., and attributes like required, placeholder, autofocus, etc.
