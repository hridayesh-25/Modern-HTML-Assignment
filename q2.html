HTML entitiies

HTML entities are special codes used to represent characters that have special meanings in HTML, or characters that cannot be easily typed on a keyboard. They are used to ensure that these characters are correctly displayed in web pages and are interpreted correctly by web browsers.

5 commonly used HTML entitities

  1. < - represents the symbol less than, with entity name &lt; and entity value &#60;. It is used todisplay the less-than sign as regular text in HTML documents without being interpreted as the beginning of an HTML tag.
  2. > - represents the symbol greater than, with entity name &gt; and entity value &#62;. It is used to display the greater-than sign as regular text in HTML documents without being interpreted as the end of an HTML tag.
  3. & - represents the symbol ampersand, with entity name &amp; and entity value &#38;. It is used to display the ampersand as regular text in HTML documents without being interpreted as the beginning of an HTML entity.
  4. " - represents the symbol double quotaion mark, with entity name &quot; and entity value &#34;. It is used to display double quotation marks as regular text in HTML documents, especially within attribute values where using double quotes would otherwise cause parsing issues.
  5.   - represents the symbol non-breaking space, with entity name &nbsp; and entity value &#160;. It is used to insert a space that is not collapsed by the browser, ensuring that text remains on the same line and preventing line breaks at that position.
